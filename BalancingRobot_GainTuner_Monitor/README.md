# Command Center For Balancing Robot






## Functions
1. GainTuner

    - 원격으로 로봇 제어기의 게인을 튜닝

2. Monitor

    - 원격으로 로봇의 상태(각도, 위치) 모니터링

3. Controller

    - 원격으로 로봇의 전진, 후진 주행과 시계, 반시계방향 회전 주행
        - 키보드의 화살표 키만 지원, 마우스로는 불가
